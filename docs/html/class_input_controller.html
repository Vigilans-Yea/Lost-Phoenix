◆ updateInput()

void InputController::updateInput ( )

从键盘更新输入。该函数会被World::update调用。

此方法将遍历键盘按键消息队列，在状态数组中将对应按键状态设为true，同时将未搜索到按键的状态设为false。

时间复杂度最差为k*n, k为按键消息队列size，n为按键枚举的size。
