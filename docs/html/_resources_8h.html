切割材质。

从材质中切割一块矩形子材质。

基准点为矩形自材质左上角坐标。

参数
texture待切割的材质。
x被切割材质基准点x坐标。
y被切割材质基准点y坐标。
w欲切割下来的宽度。
h欲切割下来的高度。
返回
切割后的材质。
