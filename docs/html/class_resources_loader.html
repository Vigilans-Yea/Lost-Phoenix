Texture ResourcesLoader::loadTexture ( const wchar_t *  imagePath,
int  w_raw,

int  w_to = 0,

)

通过元信息加载材质。

当w_to或h_to中有任意一项为0时，表示不缩放。

参数
imagePath材质的路径，采用Unicode。
w_raw材质的原分辨率宽度。
h_raw材质的原分辨率高度。
w_to(可选) 期望缩放到的分辨率宽度。
h_to(可选) 期望缩放到的分辨率高度。
返回
加载成功后，返回加载而得的材质。
