进行一次射击。高等级时子弹移动速度出现变化。 

随着difficulty\+Level升高，开始表现出不同的特性：
\begin{DoxyItemize}
\item 难度等级到2以上后，子弹的速度将在一个区间变动，初始为0.8$\sim$1.2倍原速度。随着难度越来越高，区间将变得越来越宽。
\item 这样做会使得小飞机构造出一个子弹网，使得玩家飞机难以通行。 
\end{DoxyItemize}
